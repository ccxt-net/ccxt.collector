## 작성 규칙

- 파일명: `x.y.z.md`
- 헤더: `# x.y.z - YYYY-MM-DD`
- 섹션(필요한 것만 사용):
  - Added
  - Changed
  - Fixed
  - Removed
  - Performance
  - Security
  - Migration
- 날짜가 미정인 경우 초안은 `# x.y.z (Unreleased)`로 시작합니다.
